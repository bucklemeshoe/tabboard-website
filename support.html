FAQ

How do I install TabBoard?

TabBoard is available as a Chrome extension. Here's how to install it:

  • Visit the Chrome Web Store and search for "TabBoard"
  • Click "Add to Chrome" and confirm the installation
  • Pin the extension to your toolbar for easy access
  • Grant necessary permissions when prompted
Is my data safe and private?

Yes, absolutely! TabBoard is designed with privacy as a core principle:

  • All data stays completely local on your device
  • No external servers or cloud storage
  • No tracking, analytics, or data collection
  • Your browsing sessions remain private and secure
How many tabs can I save in a session?

There's no hard limit on the number of tabs you can save in a session. However, for optimal performance:

  • Sessions with 50-100 tabs work smoothly
  • Very large sessions (200+ tabs) may take longer to restore
  • Consider breaking large projects into smaller, focused sessions
  • Regular cleanup of old sessions helps maintain performance
Can I use TabBoard on multiple devices?

Yes! TabBoard includes import/export functionality for cross-device sync:

  • Export your sessions as JSON files
  • Transfer files to other devices via cloud storage or email
  • Import sessions on your other devices
  • Data merges safely without overwriting existing sessions
What happens when I restore a session?

When you restore a session, TabBoard:

  • Opens all saved tabs in a new Chrome window
  • Maintains the original order of tabs
  • Preserves tab titles and URLs
  • Shows a visual animation during restoration
  • Keeps your current tabs untouched
How do I update an existing session?

To update a session with new tabs:

  • Open the session you want to update
  • Add or remove tabs as needed
  • Click "Update" next to the session name in TabBoard
  • The session will be refreshed with current tabs
Can I customize the keyboard shortcut?

Yes! You can customize TabBoard's keyboard shortcut:

  • Go to Chrome's Extensions page (chrome://extensions/)
  • Click the menu icon (three lines) in the top-left
  • Select "Keyboard shortcuts"
  • Find TabBoard and set your preferred shortcut
Why aren't my sessions saving?

If sessions aren't saving, try these troubleshooting steps:

  • Make sure you're not in Chrome's incognito mode
  • Check that TabBoard has permission to access tabs
  • Verify you have sufficient storage space
  • Try refreshing the extension in Chrome's extensions page
  • Restart Chrome and try again
How do I back up my sessions?

TabBoard makes it easy to back up your sessions:

  • Click the "Export" button in the TabBoard popup
  • Choose a location to save the JSON file
  • Store the file in your preferred cloud storage
  • Use the "Import" button to restore from backup
Is TabBoard free to use?

Yes! TabBoard is completely free to use with no subscriptions or hidden costs:

  • All features are available at no cost
  • No premium tiers or paid upgrades
  • No advertisements or sponsored content
  • Built for the community by tab management enthusiasts
