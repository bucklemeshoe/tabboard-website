Footer Component Example

This page demonstrates how easy it is to use the reusable footer component.

Simply include <script src="components/footer-include.js"></script> in your HTML.

How it works

The footer component automatically injects the footer HTML and CSS into your page when the script loads.

No manual HTML copying or CSS linking required!

Requirements

Your page just needs to have the required CSS variables defined (like this page does).

The footer will automatically appear at the bottom of the page.
